This project is a full-stack BnB clone web application built with Node.js, Express, MongoDB, and EJS, designed to replicate the core features of platforms like Airbnb. It enables users to seamlessly add, edit, and manage their own property listings, while also allowing customers to browse and view BnBs from all over.

  • User Authentication for owners and customers
  • Listing Creation with title, description, price, location, and images
  • Edit/Delete options for owners to manage their BnBs
  • Customer view with detailed listing pages
  • Responsive interface using EJS and Bootstrap
System Requirements
  • Processor: Intel Core i3 or higher
  • RAM: Minimum 4GB
  • OS: Cross-platform (Windows/Linux/macOS)
  • Dependencies: Node.js, MongoDB, Cloudinary (for image upload)
Current Capabilities
  • Create and manage personal BnB listings as an owner
  • Dynamic rendering of listing data with EJS templates
  • Secure routes and access control using Express middleware
  • Cloudinary integration for storing and displaying images
Future Upgrades
  • Search & Filter functionality for listings
  • Booking system with calendar integration
  • Review & rating system for customer feedback
  • Map integration using Mapbox or Google Maps API
